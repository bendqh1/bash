## Keyboard shortcuts

    # Move to the start of line
    Ctrl + a
    
    # Move to the end of line
    Ctrl + e
    
    Move forward a word (a word contains alphabets and digits, no symbols)
    Meta + f
    
    # Move backward a word
    Meta + b
    
    # Clear the screen
    Ctrl + l

## Commands

    ${BASH_VERSION}
    zip -r ./seo-wiki.zip ./seo-wiki.org/

## Notes

    # "meta" can be "F"
